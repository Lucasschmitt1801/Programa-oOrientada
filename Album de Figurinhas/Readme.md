# Repositório de Exercícios - Programação Orientada a Objetos

Este repositório contém as soluções para as listas de exercícios da disciplina de Algoritmos e Programação: Orientação a Objetos da UNISINOS.

## Organização

O repositório está organizado em diretórios, um para cada lista de exercícios:

- **/Lista1**: Exercícios de introdução a classes e objetos.
- **/Lista2**: Exercícios de modelagem e implementação de classes.
- **/Lista3**: Exercícios sobre alocação dinâmica.
- **/Lista4**: Exercício sobre persistência de dados e arquivos.

Cada diretório de exercício contém seus próprios arquivos-fonte (.h, .cpp) e um `main.cpp` para testes.
